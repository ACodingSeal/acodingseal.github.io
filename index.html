Grass Cutting Incremental Wiki

  • Astral Calculator (temporarily unavailable): Tool for calculating Space Power and time requirements to go from one Astral to another. Planned changes: Astral Bonuses support.
  • Eclipse Calculator: Tool for calculating Solar Ray and time requirements to from one Eclipse to another.
  • Solarian Stage Calculator: Tool for calculating Fighting Stage requirements and Stage Bonuses.
  • Suffix Converter (temporarily unavailable): Tool for easily converting between scientific notation and suffix notation, using the game's suffixes. Planned changes: Changing the input to case-insensitivity and other minor optimisations.
  • Star Tier Calculator (planned): Tool for calculating Star Growth and Eclipse requirements for a given Star Tier.
  • Tier Calculator (planned): Tool for calculating TP and time requirements, as well as effect differences (Grass/XP bonus, SP multiplier), to go from one tier to another.
  • Upgrade Calculator (planned): Tool for calculating costs and effect differences (e.g. 2x/+100% > 3x/+200% = 1.5x/+50%) of a level of an upgrade to another level of the same upgrade. Planned changes: Comparison of two or more upgrades that each affect the same thing.
